# Changelog

All notable curated-data, website, and automation changes are documented here.

## 0.1.0 — 2026-07-20

- Initialized a bilingual, domain-first directory with 101 curated projects.
- Added 21 stable foundation and application categories.
- Added a searchable, filterable, static Astro website with English and Chinese routes.
- Added Zod and JSON Schema validation, generated READMEs, tests, link checks, and site export.
- Added candidate discovery and GitHub metadata synchronization scripts.
- Added CI, contribution forms, and a weekly Codex update pull-request workflow.
