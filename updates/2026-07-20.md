## Excluded during first review

Common exclusions were forks and mirrors, course copies, generic agent lists, ordinary single-agent assistants, generic RAG, projects whose only multi-agent evidence was a name, low-content demos, and repositories without a clear license or primary implementation.

## Validation

Candidate text was treated as untrusted input. No candidate code, commands, dependencies, or binaries were executed. Data validation, generated README consistency, unit tests, type checks, lint, static site build, and link checks are recorded in `docs/BUILD_PLAN.md`.
